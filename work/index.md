## Virtual Home Configurator
**Personal Project · Unity · C# · 3D Visualization**

An interactive 3D home configuration application that lets users design and pre-visualize their living space before committing to a purchase. Built entirely in Unity as a personal project to explore real-time 3D product visualization and UX design for non-technical end users.

**Key Features:**
- 🏠 **Room Layout Design** — users define their floor plan and configure room dimensions in a real-time 3D environment
- 🛋️ **Furniture Placement** — drag-and-drop furniture catalog with free positioning, rotation, and scaling inside the virtual space
- 👁️ **Pre-Visualization** — walk through the configured space from a first-person or orbit camera perspective to see how furniture and layouts feel before buying
- 🛒 **Order Integration** — once satisfied with the design, users can proceed to place an order directly from the configured scene
- 🎨 **Material & Style Variants** — swap finishes, colors, and fabric options on furniture pieces in real time

**Tech Stack:** Unity 3D · C# · Custom UI system · 3D asset pipeline
